# Environment Setup

## Requirements
- Python 3.7 or later
- A text editor or IDE (VS Code recommended)
- Git (optional, for version control)

## Installation Steps

### Windows
1. Download Python from python.org
2. Run installer, CHECK "Add Python to PATH"
3. Open Command Prompt, verify: `python --version`

### macOS/Linux
1. Open terminal
2. Install: `sudo apt install python3` (Linux)
3. Or download from python.org
4. Verify: `python3 --version`

## Required Libraries
Most lessons use built-in modules. For NumPy (Lesson 15):
\`\`\`bash
pip install numpy
\`\`\`

## Test Your Setup
\`\`\`bash
python -c "print('Python works!')"
python -c "import statistics; print('Built-ins good')"
python -c "import numpy; print('NumPy works!')"
