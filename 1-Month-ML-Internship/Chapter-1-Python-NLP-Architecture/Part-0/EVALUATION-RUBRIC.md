# Evaluation Rubric

## For Self-Assessment

### Variables & Types (Lesson 1)
- [ ] Define variables with meaningful names
- [ ] Understand all basic types (int, float, str, bool)
- [ ] Convert between types correctly
- [ ] Use type() to check data types
**Mastery Score:** 1 (struggling) → 5 (expert)

### Strings (Lesson 2)
- [ ] Index and slice strings
- [ ] Use string methods (.lower, .split, .replace, etc.)
- [ ] Format strings with f-strings
- [ ] Process text data
**Mastery Score:** 1 → 5

### Numbers & Math (Lesson 3)
- [ ] Perform arithmetic operations
- [ ] Use math module functions
- [ ] Understand integer vs float division
- [ ] Solve mathematical problems
**Mastery Score:** 1 → 5

### Lists (Lesson 4)
- [ ] Create and modify lists
- [ ] Use list methods (append, insert, remove)
- [ ] Iterate through lists
- [ ] Slice lists efficiently
**Mastery Score:** 1 → 5

### Dictionaries (Lesson 5)
- [ ] Create dictionaries
- [ ] Access and modify values
- [ ] Use dict methods (.keys, .values, .items)
- [ ] Iterate through key-value pairs
**Mastery Score:** 1 → 5

### Conditionals (Lesson 7)
- [ ] Write if/elif/else statements
- [ ] Use comparison operators correctly
- [ ] Combine logical operators (and, or, not)
- [ ] Handle edge cases
**Mastery Score:** 1 → 5

### Loops (Lesson 8)
- [ ] Write for loops
- [ ] Write while loops
- [ ] Use break and continue
- [ ] Nested loops
**Mastery Score:** 1 → 5

### Functions (Lesson 9)
- [ ] Define functions with parameters
- [ ] Return values
- [ ] Use default parameters
- [ ] Write reusable code
**Mastery Score:** 1 → 5

### File I/O (Lesson 11)
- [ ] Read files
- [ ] Write files
- [ ] Append to files
- [ ] Use with statement
**Mastery Score:** 1 → 5

### Error Handling (Lesson 12)
- [ ] Use try/except blocks
- [ ] Handle specific exceptions
- [ ] Use finally block
- [ ] Write robust code
**Mastery Score:** 1 → 5

---

## Capstone Evaluation
**Excellent (90-100%):** All 10 skills at level 4-5
**Good (80-89%):** 8+ skills at 3+, others 2+
**Acceptable (70-79%):** 6+ skills at 3+
**Needs Work (<70%):** Multiple skills below 3
